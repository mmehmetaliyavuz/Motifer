STREME Integration (Important Notice)

STREME is part of the MEME Suite, which currently has severe dependency issues on macOS (especially Apple Silicon, ARM64). As a result, the local installation of STREME may fail due to missing Perl, ICU, or OpenMPI builds in the conda ecosystem.

To ensure reproducibility and platform-independent execution, the STREME module of the Motifer pipeline is provided as an external Google Colab notebook.

Run the notebook to obtain:

This approach guarantees that STREME works identically on all systems (macOS, Windows, Linux) without local installation.
